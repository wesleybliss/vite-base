## Setup

Clone the project

```shell
$ git clone https://github.com/wesleybliss/vite-base app && cd $_
```

Run the setup script (optional)

```shell
$ ./setup.sh app-slug 'App Name'
```

Install dependencies & start the dev server

```shell
$ yarn
$ yarn dev
```
